# คู่มือการใช้งานระบบสมาชิก (Membership System) - ร้านหมาล่าเชิงดอย

ระบบสมาชิกนี้ถูกออกแบบมาเพื่อตอบแทนท่านลูกค้าประจำ โดยเน้นความง่ายและความคุ้มค่าในการใช้งาน

## 1. วิธีการสะสมแต้ม (Earn Points)
- ทุกครั้งที่ท่านมียอดใช้จ่าย ท่านจะได้รับแต้มสะสม **10% ของยอดรวม**
- ตัวอย่าง: ทานหมาล่าครบ 500 บาท เมื่อชำระเงินท่านจะได้รับ 50 แต้มทันที
- แต้มจะถูกคำนวณจากยอดที่ชำระจริง (หลักหักส่วนลดอื่นๆ แล้ว)

## 2. วิธีการใช้แต้ม (Redeem Points)
- ท่านสามารถนำแต้มสะสมมาใช้เป็นส่วนลดเงินสดได้
- อัตราการแลก: **1 แต้ม = ส่วนลด 1 บาท** (ไม่มีขั้นต่ำ)
- ท่านสามารถแจ้งพนักงานเมื่อต้องการใช้แต้มเพื่อรับส่วนลดในบิลนั้นๆ

## 3. การตรวจสอบแต้ม (Point Dashboard)
- ท่านสามารถล๊อกอินเข้าระบบเพื่อดูแต้มสะสมปัจจุบันและประวัติการแลกแต้มได้ตลอดเวลา

# Membership System Technical Implementation (Backend API)

ระบบถูกสร้างขึ้นบนโครงสร้างดังนี้:
1. **User Database**: เก็บข้อมูล `pointTotal` เพื่อติดตามแต้มปัจจุบันของผู้ใช้
2. **Bill Service**: คำนวณยอดเงิน (price), ส่วนลด (discount), และแต้มที่ได้รับ (point) แบบอัตโนมัติในขั้นตอนการออกบิล
3. **Transaction Logs**: บันทึกทุกกิจกรรมที่มีผลกับแต้ม โดยระบุประเภทเป็น `EARN` (สะสม) หรือ `REDEEM` (ใช้แต้ม) เพื่อความโปร่งใส
